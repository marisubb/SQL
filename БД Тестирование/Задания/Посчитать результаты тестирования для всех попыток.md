# Посчитать результаты тестирования для всех попыток

## Задание

Посчитать результаты тестирования. Результат попытки вычислить как количество правильных ответов, деленное на 3
(количество вопросов в каждой попытке) и умноженное на 100. Результат округлить до двух знаков после запятой. 
Вывести фамилию студента, название предмета, дату и результат. Последний столбец назвать Результат. Информацию отсортировать сначала по фамилии студента, потом по убыванию даты попытки.

## Логической схемы базы данных

![Исправленная ЛС БД Тестирование](https://github.com/user-attachments/assets/eb759038-38df-47c9-83ef-756e1120c077)


## Решение

```mysql
SELECT name_student, name_subject, date_attempt, ROUND(SUM(is_correct)/3*100, 2) AS Результат
FROM student INNER JOIN attempt USING(student_id)
    INNER JOIN testing USING(attempt_id)
    INNER JOIN answer USING(answer_id)
    INNER JOIN subject USING(subject_id)
GROUP BY name_student, name_subject, date_attempt
ORDER BY name_student ASC, date_attempt DESC
```
