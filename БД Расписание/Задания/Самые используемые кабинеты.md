## Решение

```mysql
SELECT classroom
FROM Schedule
GROUP BY classroom
HAVING COUNT(number_pair) = (
		SELECT COUNT(number_pair)
		FROM Schedule
		GROUP BY classroom
		ORDER BY 1 DESC
		LIMIT 1
	)
```
